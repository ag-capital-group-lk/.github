# AG Capital Lanka (Pvt) Ltd

Welcome to the official GitHub organization for **AG Capital Lanka (Pvt) Ltd**.

This organization hosts our internal software projects, tools, and infrastructure code. Most repositories here are private and intended for internal company use.

## About Us

AG Capital Lanka (Pvt) Ltd is a Sri Lanka-based company under the AG Group. We use this organization to manage our engineering work in a secure, organized, and collaborative way.

- 📍 Location: Sri Lanka
- 📧 Contact: github@aggroups.lk

## How We Work

- All active development happens in private repositories with branch protection enabled on `main`.
- Code changes go through pull requests with required reviews before merging.
- Access is managed through teams, not individual repository invites.
- Two-factor authentication is required for all organization members.

## Contact

For access requests or questions about this organization, please reach out to an organization owner or email **github@aggroup.lk**.
